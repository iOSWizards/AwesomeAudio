## Installation

AwesomeAudio is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'AwesomeAudio', git: 'https://github.com/iOSWizards/AwesomeAudio', tag: '0.1.0'
```
